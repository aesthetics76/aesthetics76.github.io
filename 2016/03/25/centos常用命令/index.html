文件、目录操作命令

复制命令cp

  • cp -r folder folder_bak 复制指定目录到当前目录并重命名（包括其子目录）
  • cp -r test/test_in .复制指定目录的子目录到当前目录，注意后面那个点
  • cp -a test test_bak 功能和第一个一样，但会把目录的属性也复制过来

移动命令mv

  • mv test.html test 移动test.html到test目录，原来目录中的test.html被删除
    若想移动多个文件，中间用空格隔开即可，也可使用通配符移动一类文件如”*.html”
  • mv test test_bak 移动test目录到test_bak目录中，原来的目录也会删除
  • mv test test2 将test重命名为test2，注意test2原本是不存在的

查找命令find

  • find / -name httpd.conf 查找文件名为apache配置文件，可以带通配符什么的

查看文件命令cat

与进程有关的命令

查看端口使用情况

  • netstat -apn | grep 8083 查看8083端口占用情况，包含uid信息
    查看进程所对应uid
  • ps -ax | grep httpd 查看apache的进程
    杀进程
  • kill -9 21450 杀掉uid等于21450的进程

权限命令

更改权限命令chmod

  • 数字表示形式：chmod abc file a,b,c各为一个数字，分别代表User、Group、Other三个用户组的权限
    r(可读) = 4 ，w(可写) = 2，x(可执行) = 1
    常写的一个chmod 777 -R test 让这个目录权限全开（貌似有点危险）
    -R代表递归，即目录及其子目录全部权限变更
